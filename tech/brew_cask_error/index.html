brew cask 安裝軟體時跳出權限錯誤

2015 Dec 28
brew
1 min read

通常這狀況都是出現在 OSX 10.11 ，因為系統對於根目錄的權限有做調整，因此輸入下列指令來讓目前使用者有權限存取就可以了

$ sudo chown -R $(whoami):admin /etc/Caskroom
